# 🚀 CodeSage

> **AI-Powered Code Review & Learning Platform**

CodeSage is a full-stack AI-powered platform that helps developers write better code by providing intelligent code reviews, bug detection, optimization suggestions, complexity analysis, and personalized learning insights.

The project combines modern web technologies with Google's Gemini AI to create an interactive coding mentor for students, developers, and interview aspirants.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/CodeSage.git

cd CodeSage
```

---

## Frontend

```bash
cd client

npm install

npm run dev
```

---

## Backend

```bash
cd server

npm install

npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **server** directory.

```env
PORT=5000

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=codesage

JWT_SECRET=your_secret

GEMINI_API_KEY=your_api_key

GEMINI_MODEL=gemini-2.5-flash
```
